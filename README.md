# Engenharia de Dados e Big Data - CESAR School 

Este repositório armazena os projetos e atividades práticas desenvolvidos na disciplina de Engenharia de Dados e Big Data. O foco principal é a construção de pipelines de dados (ETL), modelagem relacional e consumo de APIs.

---

## Atividades 

* **[Atividade 01: Extração e Persistência de Dados](./Atividade_01/)**
    * **Descrição:** Consumo da API REST Hipolabs para extração de dados de universidades.
    * **Foco:** Automação de coleta, tratamento de nulos, idempotência e persistência em SQLite.

---

## Stack tecnológica

* **Linguagem:** Python 3.13+
* **Banco de Dados:** SQLite
* **Gestão de Dependências:** `pip` e ambientes virtuais (`venv`)
* **Controle de Versão:** Git / GitHub
